How to build and deploy the rest API
Download the project to the local repos from the GIT url and follow the below guidelines.

1.Open windows powershell and navigate to API-Project location using "cd API-Project location"
2.To build the API: dotnet build
3.To run the API: dotnet run

Once the API is deployed, follow the below steps to run the specFlow-Gherkin automated test cases

1.Open another powershell window and navigate to API-TestProject location using "cd API-TestProject"
2.To build the API: dotnet build
3.To run the test cases: dotnet test
